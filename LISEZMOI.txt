COMMENT CA MARCHE :
  - Le panier et les favoris sont synchronises sur TOUTES les pages
    (localStorage ccl_cart / ccl_wishlist)
  - Ajouter au panier marche depuis n'importe quelle page
  - Cliquer un produit ouvre sa fiche : index.html#/produit/ID
  - Panier / commande / paiement : tout passe par index.html
  - Chaque page a sa propre adresse indexable par Google :
    votre-site/mode.html, votre-site/bestfoods.html, etc.

  ESPACE ADMINISTRATION :
    admin.html        panneau d'administration SANS CODE :
                      - ajouter / modifier / dupliquer / supprimer des produits
                      - uploader les photos directement dans le depot
                      - changer bandeau d'annonce, WhatsApp, Wave, Orange Money
                      - bouton "Publier sur GitHub" = mise en ligne automatique
                      Acces : votre-site/admin.html (non reference par Google,
                      inutilisable sans votre jeton GitHub personnel)

  MARKETPLACE VENDEURS (7500 FCFA/mois) :
    vendeur.html      espace vendeur public :
                      - offre + inscription (dossier WhatsApp + paiement
                        Wave/OM du 1er mois)
                      - formulaire "Ajouter un produit" (envoi structure
                        sur votre WhatsApp, photos jointes dans le chat)
                      - renouvellement d'abonnement
    Dans admin.html, onglet "Vendeurs" :
                      - creer la boutique apres reception du paiement
                        (code V-001 genere automatiquement)
                      - bouton "+1 mois" a chaque paiement de 7500
                      - produits d'un vendeur suspendu ou expire =
                        MASQUES AUTOMATIQUEMENT du site (pas supprimes)
                      - dans la fiche produit : champ "Vendeur" pour
                        rattacher le produit a une boutique
    Sur le site : mention "Vendu par [Boutique]" sur les cartes et
    fiches, et nom de la boutique dans les commandes WhatsApp.
