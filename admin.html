Paiements

Votre jeton reste uniquement dans ce navigateur (localStorage). Il n'est envoyé qu'à l'API officielle GitHub. Ne le partagez jamais.

Connexion au dépôt

Comment le créer : GitHub → votre photo → Settings → Developer settings → Fine-grained tokens → Generate new token → dans "Repository access" choisissez uniquement votre dépôt → dans "Permissions" → Repository permissions → Contents : Read and write → Generate. Copiez le jeton ici.

Sans connexion GitHub ?

Vous pouvez travailler hors connexion : faites vos modifications puis téléchargez le fichier products.js et uploadez-le vous-même sur GitHub (Add file → Upload files → remplacer).

Comment publier une modification

  1. Connectez GitHub une seule fois (onglet Connexion GitHub).
  2. Ajoutez / modifiez / supprimez vos produits, ou changez les paramètres.
  3. La barre du bas indique les modifications en attente → cliquez Publier sur GitHub.
  4. Attendez 1 à 2 minutes : GitHub Pages met le site à jour automatiquement.

Photos des produits

  1. Dans la fiche produit, cliquez Uploader une photo : elle part directement dans votre dépôt GitHub et le nom de fichier se remplit tout seul (connexion GitHub requise).
  2. Ou tapez simplement le nom d'un fichier déjà présent dans le dépôt (ex : montre-luxe.jpg).
  3. Les vidéos .mp4 sont acceptées comme visuel : elles jouent en boucle sur le site.

Bon à savoir

  1. Cette page n'est pas référencée par Google (noindex) et n'apparaît nulle part sur le site public. Elle est toutefois accessible à qui connaît l'adresse admin.html — sans votre jeton GitHub, personne ne peut rien modifier.
  2. Chaque publication crée un commit propre dans votre dépôt : vous pouvez toujours revenir en arrière depuis GitHub (History).
